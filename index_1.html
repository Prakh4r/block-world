Blocks World is one of the most studied domains in AI planning. The task is to rearrange a set of blocks on a table using a robotic arm with these operators:

PICK-UP(x) — pick up block x from table
PUT-DOWN(x) — put block x on table
STACK(x,y) — stack x on top of y
UNSTACK(x,y) — remove x from top of y

The arm can hold at most one block. A block is clear if nothing is on top of it.
